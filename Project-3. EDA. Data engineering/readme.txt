Основные цели и задачи проекта:
-создать свою первую модель, основанную на алгоритмах машинного обучения;

-️ принять участие в соревновании на Kaggle;

- понять, как правильно «подготовить» данные, чтобы ваша модель работала лучше.

Краткая информацию о данных:
В данных пустые значения есть только в двух столбцах lat и lng, которые мы заполнили с помощью библиотеки googlemaps. 
Выбросы мы не искали, хотя надо бы было это сделать.
Это позволяет сделать вывод о том, что данные достаточно чистые.

Этапы работы над проектом:
1.Провели первичную обработку данных. 
2.Оценили количество уникальных значений для номинативных переменных.
3.Заполнили пустые значения в колонках lat и lng.
4.Провели корреляционный анализ количественных переменных
5.Отобрали не коррелирующие переменные.
6.Проанализировали номинативные переменные и устранили те, которые не влияют на предсказываемую величину (в нашем случае — на переменную reviewer_score). В нашем случае это ухудшило модель.
