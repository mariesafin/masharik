Основные цели и задачи проекта:
Определиться, какие параметры должны быть использованы в будущей модели, предсказывающей результаты госэкзамена по математике для каждого ученика школы. Для этого мы провели разведывательный анализ данных и составили отчёт по его результатам.

Краткая информацию о данных:
В данных достаточно много пустых значений, можно еще раз попробовать их заполнить иначе. 
Например, пропуски в численных столбцах заменить средним, медианой или модой, и посмотреть на результаты. 
Категориальные мы заполнили None, но можно заполнить наиболее часто встречающимися или исключить их из датафрейма, если их конкретное значение является важным.
Выбросы найдены только в столбцах с образованием отца, отношениями в семье. Но это скорее ошибка при заполнении анкет. Так же в столбце с пропусками занятий есть два выброса.
Это позволяет сделать вывод о том, что данные не достаточно чистые.

Этапы работы над проектом:
1.Провели первичную обработку данных. 
2.Посмотрели на распределение признака для числовых переменных, устранили выбросы.
3.Оценили количество уникальных значений для номинативных переменных.
4.Удалили пустые значения в номинативных переменных.
5.Провели корреляционный анализ количественных переменных
6.Отобрали не коррелирующие переменные.
7.Проанализировали номинативные переменные и устранили те, которые не влияют на предсказываемую величину (в нашем случае — на переменную score).
8.Сформулировали выводы относительно качества данных и тех переменных, которые мы будем использовать в дальнейшем построении модели.
